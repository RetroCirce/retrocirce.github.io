USS-SEP: Computational Auditory Scene Analysis with Weakly Labelled Data

In this demo page, we present the separation demos of USS-SEP.
Specifically, we present the separation demos of two movie trailers in hierarchical separation levels.
The website may take some time to load all audio files because they are very large. We recommend to use the headphone to listen to these demos.
